data-mpa-action-id="m9l43kk11chh"><span textstyle="">40年合资企业的谢幕</span></span><span mpa-font-style="m9l43kk1wfr"></span></p><p><span leaf=""><br></span></p><p><span leaf=""><span textstyle="">根据达仁堂公告，2024年，</span></span></p><ul><li><p><span leaf=""><span textstyle="">中美史克</span><span textstyle="">营收35.59 亿元人民币、</span><span textstyle="">净利润7.11亿元</span><span leaf=""><span textstyle="">中美史克的核心产品，包括芬必得（止痛药）、舒适达（口腔护理产品）、新康泰克（感冒药）等，通过中国境内的医药零售终端、电商平台以及医疗机构进行销售。</span></span></p><p><span leaf=""><br></span></p><p><span leaf=""><span textstyle="">芬必得在中国止痛药市场中长期占据前三的位置，而舒适达在抗敏感牙膏领域的市场份额超过了30%。</span></span></p><p><span leaf=""><br></span></p><p><span leaf=""><span textstyle="">此外，近年来中美史克加速了在中国区域市场的布局，在成都、西安、上海、广州等地设立了分公司，以推动产品在下沉市场的渗透。例如，2024年成立的西安分公司和成都分公司，主要负责在西部市场推广疼痛管理和口腔护理产品。</span></span></p><p><span leaf=""><span textstyle="">这场资本博弈始于2024年9月。</span></span></p><p><span leaf=""><span textstyle="">2024年9月</span><span textstyle="">，</span><span textstyle="">赫力昂通过两笔交易完成了对中美史克的股权增持</span><span textstyle="">：达仁堂转让了13%的股权，天津医药集团转让了20%的股权，赫力昂的持股比例从55%提升至88%。</span></span></p><p><span leaf=""><span textstyle="">当时的达仁堂董事长张铭芮表示，天津史克是一家卓越的公司，出售股权主要是因为合作期限已到。根据公告，天津史克的合资协议将于2025年6月到期。</span></span></p><p><span leaf=""><span textstyle="">然而，</span><span textstyle="">这场股权变动或许进一步揭示了在时代大潮下，跨国企业在全球化战略与本土市场之间必然发生的重构</span><span textstyle="">。</span></span></p><p mpa-font-style="m9l0qnk4v4p" data-mpa-action-id="m9l0qnkkpe4" data-pm-slice="0 0 []"><span leaf="" data-mpa-action-id="m9l06yc711ie"><span textstyle="">02</span></span></p><p mpa-font-style="m9l43wu1voo" data-mpa-action-id="m9l43wuf1b2m" data-pm-slice="0 0 []"><span leaf=""><span textstyle="">时代烙印</span></span></p><p mpa-font-style="m9l43wu1voo" data-mpa-action-id="m9l43wuf1b2m" data-pm-slice="0 0 []"><span leaf=""><span textstyle="">中美史克与中国医药产业的变迁</span></span></p><p><span leaf="" data-mpa-action-id="m9l1svuwb2r"><br></span></p><p><span leaf=""><span textstyle="">上世纪八十年代，“中美史克”和“西安杨森”（强生创新制药）作为改革开放后首批进入中国的外资合资药企，成为了当时中国医药市场具标志性和影响力的两大医药品牌。</span></span></p><p><span leaf=""><br></span></p><p><span leaf="">也承载了九十年代至千禧年，中国医药人才的主要输出；约60%的药企销售高管曾在西安杨森或中美史克任职，形成“杨森系”和“史克系”两大人才阵营。</span></p><p><span leaf="">1992年芬必得上市后，中美史克首创“医院端学术推广+零售端品牌营销”双轨制，培养了一批兼具专业能力与市场敏感度的药品销售人才。</span></p><p><span leaf=""><span textstyle="">而中美史克四十年发展历程，同样是</span><span textstyle="">中国医药产业对外开放与全球化合作历程的一个缩影</span><span textstyle="">。</span></span></p><p><span leaf="" data-mpa-action-id="m9l1svuw1xti"><span textstyle="">1984年，美国史克</span></span><span leaf=""><span textstyle="">必成公司（后并入葛兰素史克）与天津医药集团签订协议，在天津东丽区筹建制药厂。三年后，一座按照美国FDA标准设计的现代化制剂厂正式投产，成为中国首批引入国际GMP规范的药企之一。</span></span></p><p><span leaf=""><span textstyle="">起初，“中美史克”主要生产抗生素和消化系统药物，凭借芬必得（布洛芬缓释胶囊）、泰胃美（西咪替丁）等产品迅速占领市场。到了1998年，其口服制剂车间通过国家GMP认证，年产能力达到23亿片/粒/支，员工规模扩展至800余人。</span></span></p><p><span leaf=""><span textstyle="">在这一阶段，“中美史克”扮演了外资药企进入中国的“探路者”角色。它引入的缓释技术、市场推广理念，以及与天津医药集团的本土化合作模式，为后续跨国药企在华发展树立了典范。</span></span></p><p><span leaf=""><span textstyle="">随着中国医药市场的变革</span><span textstyle="">，“中美史克”开启了多元化战略：2010年成立中国创新中心；2015年开设天猫旗舰店，成为首家“触网”的跨国药企。</span></span></p><p><span leaf=""><span textstyle="">随着中国对外开放力度的持续加大</span><span textstyle="">，赫力昂中国早在2023年2月已启动“无痛中国行动”，组建由权威专家领衔的科普团队，通过短视频、直播等形式普及疼痛认知；2024年3月，联合全国3000家零售药店设立“疼痛管理专区”，并与美团、饿了么合作推出“无痛中国服务中心”，提供线上问诊、药品配送、康复指导等一站式服务。</span></span></p><p><span leaf=""><span textstyle="">2024年9月，《外商投资准入特别管理措施（负面清单）（2024 年版）》正式发布，根据该政策及配套政策文件，外资药企在中国设立全资子公司的政策框架已全面落地。</span></span></p><p><span leaf=""><span textstyle="">这一切似乎都恰逢其时，水到渠成。</span></span></p><p><span leaf=""><span textstyle="">针对本次收购，赫力昂表示，“这是赫力昂进入中国市场40年之后，在新时代重申对华承诺的重要里程碑时刻。增持天津史克的股权也是出于更加深入中国市场的考量。”</span></span></p><p mpa-font-style="m9l0qu4b7p0" data-mpa-action-id="m9l0qu4s17s" data-pm-slice="0 0 []"><span leaf="" data-mpa-action-id="m9l08jqf1efi"><span textstyle="">03</span></span></p><p><span leaf="" data-mpa-action-id="m9l44lc5pym"><span
