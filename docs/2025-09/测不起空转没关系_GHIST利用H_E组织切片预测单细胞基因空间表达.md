------
<div><section data-tool="mdnice编辑器" data-website="https://www.mdnice.com" data-pm-slice="0 0 []" data-mpa-action-id="mfl2ldh91h0c"><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2mikjab1" data-mpa-action-id="mfl2mil2qid" data-pm-slice="0 0 []"><span textstyle="">为什么需要用H&amp;E切片预测基因表达？</span></span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgf2iv">空间转录组学（Spatial Transcriptomics, ST）技术能够在组织切片上测量基因表达的空间分布，为理解多细胞系统提供了革命性的手段。然而，当前主流的ST平台（如10x Genomics Visium）通常以斑点（spot）为单位，每个斑点涵盖多个细胞，因此空间分辨率有限。近年来出现的亚细胞原位空间转录组（subcellular spatial transcriptomics, SST），如10x Xenium、NanoString CosMx及Vizgen MERSCOPE等，可以达到接近单细胞甚至亚细胞的分辨率。这些SST数据通常需要结合细胞分割，将信号归属于各个细胞，从而得到单细胞空间基因表达谱。尽管SST技术具有更高分辨率，但其高昂的成本和复杂的操作限制了大规模应用。与之对比，常规的组织病理切片（如苏木精-伊红染色，H&amp;E）在临床中被大量累积。如果能<span textstyle="">仅凭H&amp;E切片预测每个细胞的基因表达</span>，将为广泛可及的病理数据赋予分子信息，推动发现新的生物学模式。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgf294">为此，近年出现了多种深度学习方法尝试从H&amp;E图像预测空间基因表达。早期方法多针对斑点级的基因表达预测，例如ST-Net、Hist2ST、HisToGene、DeepPT、BLEEP等。典型做法是<span textstyle="">用卷积神经网络（CNN）或Transformer提取与每个斑点对应的图像特征，再用全连接网络输出该斑点的转录组</span>。一些方法还将相邻斑点关系纳入建模，例如Hist2ST结合了ConvMixer、Transformer和图神经网络（GNN）以利用全局空间依赖。另有策略对斑点数据进行“超分辨率”处理，将其细化为更小像素点的预估表达。然而，由于斑点可能混合多个细胞，这些方法无法真正达到单细胞精度。近期的基准研究也显示，现有方法预测精度有限，难以捕捉微观异质性。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgf6jq">GHIST（Gene expression from HISTology）应运而生，它由澳大利亚悉尼大学等研究者提出，并发表于<span textstyle="">Nature Methods</span>。GHIST的目标是在单细胞水平预测空间基因表达，并提高预测的准确性和生物合理性。接下来，我们将对GHIST的方法设计、算法原理、性能评估以及应用价值进行深入解读，并结合论文中的主要图表进行说明。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2p6terhh" data-mpa-action-id="mfl2p6u21899" data-pm-slice="0 0 []"><span textstyle="">GHIST的多任务多模态架构</span></span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgfalh">图1展示了GHIST的整体框架。GHIST以配对的亚细胞空间转录组数据（SST）和对应的H&amp;E图像为训练输入，通过深度学习模型学习从图像到单细胞基因表达的映射。GHIST的核心思想是在预测基因表达的同时，<span textstyle="">联合学习多个层次的生物学信息，包括： (i) 细胞类型, (ii) 细胞邻域组成, (iii) 细胞核形态, 以及 (iv) 单细胞RNA表达</span>。为此，GHIST构建了多任务学习架构，包含四个预测子网络（prediction heads），分别对应上述四类信息。通过多任务联合训练，模型可以同时学习不同信息层次间的关联关系。</span></p><section nodeleaf="" data-mpa-action-id="mfl2ro4165" data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgf184w">多任务架构中，各预测头的特征和预测输出相互交织利用，以捕获生物信息的依存关系。例如，<span textstyle="">GHIST使用细胞邻域组成来辅助基因表达预测，利用细胞的基因表达来预测其细胞类型，从而确保预测的表达具有生物学合理性</span>。在训练过程中，细胞类型和邻域组成由已知的SST数据获得，用作监督信号训练模型预测这些辅助任务。这样，模型被引导提取与细胞类型和微环境相关的视觉特征，提升对基因表达模式的辨识能力。此外，GHIST允许可选地输入单细胞参考转录组（如单细胞RNA测序数据）以进一步指导模型学习基因共表达等规律。值得注意的是，这些辅助信息仅在训练时需要；GHIST推断阶段只需提供待预测的H&amp;E图像，无需额外的单细胞数据。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg1f6x">GHIST的训练流程包括预处理步骤：<span textstyle="">对H&amp;E图像进行颜色标准化（stain normalization），利用已有方法对图像进行细胞核分割，进而得到细胞区域。对于训练集的每个细胞，来自SST的数据提供“真值”基因表达谱以及细胞类型标签等。模型以每个细胞的图像（通常裁剪的细胞邻域patch）为输入，经过卷积编码器（Encoder）提取特征，再通过多任务预测头输出该细胞的基因表达向量、细胞类型以及邻域细胞组成等</span>。<span textstyle="">训练损失由主任务（基因表达的回归误差，例如均方误差）和辅助任务（细胞类型分类的交叉熵、邻域组成预测误差等）共同构成</span>。多任务学习使模型的中间表示同时服务于不同任务，从而学习到图像特征与多层次生物信息的关联。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg1k46">值得一提的是，GHIST在方法设计上吸收了生物先验知识。例如，它将细胞核形态作为辅助任务之一，因为形态特征往往反映细胞的类型和状态，对基因表达有提示作用。论文中引用了Hover-Net等细胞核分割与分类模型来支持这一部分任务。GHIST利用这些先验，使模型更容易捕获“肉眼难辨的细微差异”中所蕴含的表达模式。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldggeh7">GHIST模型训练完成后，即<span textstyle="">可对任意H&amp;E染色的组织切片进行单细胞基因表达预测</span>。推理时，对输入的切片图像进行同样的颜色归一化和细胞分割，将每个细胞送入模型编码器，通过已学得的映射得到数百个基因在该细胞的表达强度估计。<span textstyle="">这些预测的单细胞空间基因表达（Single-cell Spatial Gene Expression, SGE）可以视为一种新数据模态，可与传统的组学数据和临床数据结合，进行下游分析</span>。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2vi3l1hev" data-mpa-action-id="mfl2vi46122i" data-pm-slice="0 0 []"><span textstyle="">性能评估</span></span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg14rp">作者在多种公开数据集上验证了GHIST的性能。首先，在两个高分辨率的乳腺癌SST数据集（称为BreastCancer1和BreastCancer2，对应10x Xenium平台）上，检查GHIST对单细胞水平基因表达的预测效果。然后，在一个较低分辨率的HER2+乳腺癌斑点级数据集（称为HER2ST数据集）上，与多种现有方法进行对比。最后，将GHIST应用于TCGA乳腺癌队列的H&amp;E切片，展示其在风险预测、异质性挖掘和拷贝数变异模式识别等下游任务中的价值。下面我们分步骤介绍这些结果。</span></p><p •	差异基因分析（图5b,c）：GHIST的单细胞表达数据使作者能够在特定细胞类型内比较不同患者组的基因表达差异。例如，将HER2+患者按激素受体状态分为ER+/PR+和ER−/PR−组，GHIST预测揭示在巨噬细胞和基质细胞中存在特定的差异表达基因。图5(b)列举了几个显著差异基因（如LPL、CAVIN2、TIMP4、ADIPOQ），其中一些与脂质代谢和微环境相关，在ER+/PR+组表现更高。更有趣的是，在ER+/PR+组内部，这些基因的表达呈现异质性：有一部分患者的这些基因表达普遍较高，可将ER+/PR+组进一步聚类分成两个亚组（图5c，其中“Cluster 2”为高表达亚组）。这种异质性仅通过传统分类无法发现，而GHIST提供的数据让我们看到了表型下的分子分层。进一步比较各种特征对区分这两个亚组的贡献（图5d），发现细胞类型特异的基因表达特征有最高的判别能力（平衡准确率~0.87），远胜于仅细胞组成（~0.66）或空间模式特征。这再次强调了GHIST所提供的细胞分辨率转录组在揭示细微分子差异上的价值。 • 基因组拷贝数变异（CNA）对空间表达模式的影响（图5e）：由于TCGA有对应的DNA拷贝数数据，作者探究了是否存在CNA驱动的空间表达变化。<span textstyle="">他们将每个基因的空间表达模式与基因座的CNA状态相关联，寻找显著的关联（称为“差异patterning基因”）</span>。图5(e)显示了3个选定热点区域（染色体1、8、17q）的火山图，每个点代表一个基因在有无该区域CNA时空间表达模式差异的显著性。可以看到，17q区域（包含HER2/ERBB2基因）影响了最多的基因空间模式，这与该区域在乳腺癌中高扩增的已知情况一致。尤其是TOP2A（位于17q12附近）在有无17q扩增时呈现显著的空间差异表达（统计量突出），这与TOP2A拷贝数变异已知可预测化疗响应的报道相符。此外，在另一个亚型（ER阴性）中，17号染色体同样是关联空间表达变化的“热区”，说明该染色体的CNA对乳腺癌的空间分子表型有普遍影响。这些发现佐证了GHIST预测的数据可用于将组织形态、转录组和基因组变异建立联系，帮助定位新的分子标志物。例如，<span textstyle="">从这些结果推测，某患者肿瘤如有17q扩增，GHIST预测的空间表达模式可能会呈现特定改变，这种改变或可作为预后或疗效预测的依据之一</span>。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg1w9s">综上，GHIST在TCGA独立队列上的应用展示了强大的泛化能力和探索潜力。它不仅复现了已知的肿瘤生物学规律（如8q24“基因荒漠”区域、17q扩增等对表达的影响），还提出了新的假设（如通过空间表达模式细分ER+患者亚组），这些都来源于仅凭病理图像得出的数据。这证明了GHIST有望将海量的历史病理切片变为可挖掘的分子宝库，为研究者提供成本低廉的假设生成工具。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl340tdieh" data-mpa-action-id="mfl340tx21sg" data-pm-slice="0 0 []"><span textstyle="">方法优势、局限与展望</span></span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg21n5">GHIST的优势是显而易见的。首先，它实现了前所未有的单细胞空间分辨率预测，大幅提升了从H&amp;E图像获取分子信息的能力。多任务学习使模型能整合多层次特征，因而预测准确性和生物相关性显著优于以往斑点级方法。GHIST在各种场景下展现出灵活性：无论是高分辨率的SST数据、低分辨率的Visium数据，还是无测序数据的独立切片，都能有效应用。这意味着经过充分训练后，GHIST可作为一个通用工具，为任意病理切片“赋予”空间转录组信息。对临床已有海量H&amp;E切片的再利用，将大大扩充可供分析的多组学维度，为发现新的疾病机制和生物标志物创造机会。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg11s1">GHIST的预测结果还是一种全新的数据形式，能够与基因组、转录组、蛋白质组等现有数据结合进行多视角整合分析。例如上文展示的细胞邻域相互作用、空间异质性、CNA关联等分析，就是利用GHIST数据取得的。这预示着，在未来研究中，可以把GHIST生成的单细胞SGE视作体外实验的替代，先于实际消耗资源的实验，在计算机上尝试验证一些假设。如果GHIST等方法预测出有趣的分子模式，再去设计针对性的实验，将大幅提高研究的效率。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg1mkx">当然，局限性也需要理性看待。<span textstyle="">GHIST本质上仍是预测模型，其输出不可能完美等价于真实测序数据</span>。例如，<span textstyle="">在一些低丰度基因上相关性仍较低，这可能受限于图像本身的信息量——毕竟并非所有基因的变化都能从形态学看出端倪</span>。而且GHIST的训练高度依赖于有配对ST数据的组织类型，跨组织泛化能力取决于训练集的覆盖范围。<span textstyle="">如果遇到截然不同的组织或工艺，其表现可能下降</span>。因此，随着更多高质量ST数据发布，应持续扩充GHIST的训练，涵盖更多癌种和组织类型。另外，GHIST预测的准确性目前虽相对领先，但如论文所示平均PCC只有0.16左右（spot级），这提示模型还有提升空间。可能的改进方向包括引入更强大的视觉backbone（如更深的Transformer）、利用自监督预训练获取更泛化的表征，以及融合更多多模态数据（如同时输入免疫组化图像、临床变量等）以约束预测。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldgg1k5u">值得关注的是，近期也有其他类似思路的方法涌现。例如VirtualStainer、CAMIL等利用GAN或Transformer从H&amp;E合成多重标记的图像或表达。这些都印证了一个趋势：用AI模型将廉价易得的病理图像转化为丰富的分子信息。GHIST作为其中的佼佼者，在单细胞分辨率上取得了突破，为这一跨模态融合领域树立了标杆。</span></p><p data-tool="mdnice编辑器"><span leaf="" mpa-font-style="mfl2ldggixw">展望未来，GHIST及其后继方法有望在以下方面推进：（1）纳入更多任务，例如同时预测蛋白质表达、表观遗传修饰等，使模型成为更全面的“组学翻译器”；（2）优化模型效率，减少对显存和算力的需求，推动临床部署；（3）与病理AI诊断结合，直接从预测的分子谱中寻找诊断分类依据，实现真正的形态-分子一体化诊断。</span></p><p
