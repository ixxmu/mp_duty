data-linktype="2">不要轻易尝试单细胞核转录组测序</a>，因为如果大家的snRNA-seq的操作不够好，很容易导致很惨的数据质量。</p></blockquote><p data-tool="mdnice编辑器">哪怕是大家拿到了惨不忍睹的单细胞转录组数据集，也不是完全不可以发表，比如 2022的NG的文章：《Single-cell analyses define a CRCs</strong></p><p data-tool="mdnice编辑器">文献里面其实是有两个单细胞技术：</p><ul data-tool="mdnice编辑器"><li><section>single-nuclei transcriptomes (single-nucleus RNA sequencing (RNA-seq) (snRNA-seq))</section></li><li><section>epigenomes (single-cell assay for transposase-accessible chromatin using sequencing (ATAC-seq) (scATAC-seq))</section></li></ul><p
