data-pm-slice="0 0 []"><span leaf="">在生物信息学领域，斯坦福大学推出的 </span><strong><span leaf="">Biomni</span></strong><span leaf=""> 无疑是一个令人兴奋的 AI Agent 项目。它展示了 LLM 如何自动化处理组学数据、分析文献和生成图表。</span></p><p><span leaf="">然而，对于许多想要在实际研究中尝试 Biomni 的科研人员来说，目前面临着两个尴尬的选择：</span></p><ol><li><section><strong><span leaf="">使用官网在线版</span></strong><span leaf="">：功能最全，但需要</span><strong><span leaf="">上传你的私有数据</span></strong><span leaf="">。对于涉及患者隐私（HIPAA 合规）或实验室未发表成果的数据，这往往是不可接受的。</span></section></li><li><section><strong><span leaf="">使用原版本地代码</span></strong><span leaf="">：虽然保证了数据安全，但其自带的 Gradio 界面</span><strong><span leaf="">功能较为基础</span></strong><span leaf="">，缺乏多会话管理、文件浏览等现代化交互功能，使用体验比较“硬核”。</span></section></li></ol><p><strong><span leaf="">为了解决这个问题，我在 Biomni 原版的基础上进行了 Fork 与二次开发，推出了 —— Biomni-Web。</span></strong></p><p><span leaf="">我的目标很单纯：</span><strong><span leaf="">在保留斯坦福原版强大逻辑核心的同时，为你提供一个安全、好用、现代化的本地 Web 界面。</span></strong></p><hr><h3><span><span leaf="">核心改进：不仅仅是“能用”，更要“好用”</span></span></h3><p><span leaf="">Biomni-Web 不是对底层算法的重构，而是对</span><strong><span leaf="">交互体验</span></strong><span leaf="">的全面补全。我重点解决了以下痛点：</span></p><h4><span><span leaf="">1. 🛡️ 彻底的本地化 vs 在线上传</span></span></h4><p><span leaf="">不再需要纠结“数据传还是不传”。</span></p><p><span leaf="">Biomni-Web 专为本地环境设计，你的 CSV、BAM、VCF 文件直接在本地读取。AI 仅负责逻辑推理和代码生成，</span><strong><span leaf="">原始数据绝不离线</span></strong><span leaf="">。</span></p><h4><span><span leaf="">2. 💻 现代化 UI vs 原版简易界面</span></span></h4><p><span leaf="">原版的本地界面主要用于基础演示。而 Biomni-Web 引入了完整的</span><strong><span leaf="">工作流管理</span></strong><span leaf="">：</span></p><ul><li><section><strong><span leaf="">多会话并行</span></strong><span leaf="">：你可以同时开启“单细胞分析”和“文献综述”两个窗口，互不干扰。</span></section></li><li><section><strong><span leaf="">持久化记忆</span></strong><span leaf="">：网页关闭后，历史记录依然保留，随时回来接着跑分析。</span></section></li><li><section><strong><span leaf="">富媒体渲染</span></strong><span leaf="">：支持 Markdown、代码高亮以及 Plotly/Matplotlib 图表的实时渲染。</span></section></li></ul><h4><span><span leaf="">3. 📂 可视化文件管理 vs 命令行操作</span></span></h4><p><span leaf="">在原版中，生成的文件可能需要你去文件夹里翻找。Biomni-Web 内置了</span><strong><span leaf="">沙盒文件浏览器</span></strong><span leaf="">：</span></p><ul><li><section><span leaf="">每个会话拥有独立的工作目录。</span></section></li><li><section><span leaf="">AI 生成的 Excel 表格、PDF 报告、图片，直接在网页右侧点击即可预览或下载。</span></section></li></ul><hr><h3><span><span leaf="">🛠️ 功能特性对比</span></span></h3><table><thead><tr><th><section><span leaf="">功能</span></section></th><th><section><span
