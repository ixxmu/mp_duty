leaf="">作为一只资深的R语言讲师，我们做富集分析从来都是ClusterProfiler伺候。因为与R语言其他分析的完美衔接，也因为Y叔超强的开发能力和独特的人格魅力。</span></p><p data-tool="mdnice编辑器"><span data-tool="mdnice编辑器"><span leaf="">剧透一下，我们又在筹备新课了，是一篇13分的文章复现。其中用到了Metascape，觉得还不错，所以写文介绍一下它。</span></p><p data-tool="mdnice编辑器"><span leaf="">Metascape 于 2019 年在 </span><em><span leaf="">Nature Communications</span></em><span leaf=""> 发表，文章题为 "Metascape provides a biologist-oriented resource for the analysis of systems-level datasets"，是很受欢迎的富集分析工具，在CNS文章中也有应用。</span></p><h4 data-tool="mdnice编辑器"><span><span></span></span><span><span></span><span leaf="">Metascape的优点</span></span><span><span></span></span></h4><p data-tool="mdnice编辑器"><strong><span leaf="">1.简单易用</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">将基因列表粘贴到输入框，点击“Express Analysis”按钮，即可一键启动全套分析流程。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">2.月度更新</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">Metascape 坚持对其集成的 GO、KEGG、Reactome 等 40 多个主流知识库进行月度更新，确保用户的分析结果永远基于最新的科学发现，避免了许多工具因常年不维护而产生误导性结果的风险。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">3. 支持多列表 Meta 分析</span></strong><span leaf=""><br></span></p><p data-tool="mdnice编辑器"><span leaf="">Metascape 的名称来源于 "Meta-analysis"（元分析）。它支持同时分析多个基因列表，通过直观的热图（Heatmap）展示不同实验组之间共享的与特有的生物通路。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">4. 发表级的输出图表</span></strong><span leaf=""><br></span><span leaf="">Metascape 的分析报告完全模仿科研论文的格式，图文并茂，并详细阐述了分析方法。</span></p><p data-tool="mdnice编辑器"><span leaf="">图片提供pdf、png以及画图数据， 表格提供</span><strong><span leaf="">Excel</span><strong><span leaf=""><span textstyle="">格式，蛋白互作网络文件提供</span></span></strong><span leaf="">cys</span></strong><span leaf="">格式，还有个丑萌的</span><strong><span leaf="">ppt</span></strong><span leaf="">。可以一键</span><strong><span leaf="">打包下载</span></strong><span leaf="">。</span></p><h3 data-tool="mdnice编辑器"><span></span><span><span leaf="">1.通路富集分析</span></span><span></span></h3><h4 data-tool="mdnice编辑器"><span><span></span></span><span><span></span><span leaf="">Figure 1. Bar Graph Summary</span></span><span><span></span></span></h4><p data-tool="mdnice编辑器"><span leaf="">条形图，展示了多个数据库的去除冗余后的富集结果。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-src="https://mmbiz.qpic.cn/sz_mmbiz_png/2DGSDzibQdKsEOTnvf4H13pACj8vIQP4vKwjGrCrlgxkoRdNRXvBv81OUuPwfZhk70hbbGtXUv09dFXKYtiansrFibw2NzFsbiaXHyFHlFZX008/640?wx_fmt=png&amp;from=appmsg" data-ratio="0.4212121212121212" data-type="png" data-w="990" data-imgfileid="100013827" data-aistatus="1" src="https://mmbiz.qpic.cn/sz_mmbiz_png/2DGSDzibQdKsEOTnvf4H13pACj8vIQP4vKwjGrCrlgxkoRdNRXvBv81OUuPwfZhk70hbbGtXUv09dFXKYtiansrFibw2NzFsbiaXHyFHlFZX008/640?wx_fmt=png&amp;from=appmsg"></span></figure><p data-tool="mdnice编辑器"><strong><span leaf="">1. GO 开头的条目（例如 GO:0045071）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 Gene Ontology (GO) 数据库。这是一个生物信息学领域最基础的资源，致力于建立一套标准的词汇表来描述基因产物的特性。Metascape 的富集分析主要侧重于其中的生物学过程（Biological Process），用于描述基因参与的广泛生命活动，如细胞周期、免疫反应等。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">2. R-HSA 开头的条目（例如 R-HSA-9909505）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 Reactome 数据库。这是一个经过专家人工审阅的开源生物通路数据库。与 GO 相比，它更侧重于详细的分子反应流程，展示信号转导和代谢反应的具体步骤，通常能提供比 GO 更具体的分子机制信息。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">3. WP 开头的条目（例如 WP5218）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 WikiPathways 数据库。这是一个由社区维护、开放协作的生物通路数据库。它的特点是更新速度极快，往往能第一时间收录最新的疾病相关通路（如 COVID-19 相关机制），弥补了传统数据库更新滞后的短板。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">4. hsa 开头的条目（例如 hsa05170）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 KEGG Pathway 数据库。这是一个经典的生物通路数据库，以人工绘制的通路图闻名。它将基因组信息与功能信息（如代谢、细胞过程、人类疾病）联系起来，非常适合用于理解基因在上下游信号网络中的位置。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">5. M 开头的条目（例如 M255）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 MSigDB (Molecular Signatures Database) 中的 Canonical Pathways 集合。这是一个整合型数据库，汇聚了来自 PID、BioCarta 等多个来源的经典信号通路。它主要用于识别已经被广泛验证的信号级联反应和转录因子调控网络。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">6. M 开头的条目（例如 M41711）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">在“Cell Type Signatures”分析板块中，M 开头的 ID 对应 MSigDB C8 集合，即细胞类型特征数据库。这是一个基于大量单细胞测序数据构建的资源。它收录了特定细胞类型的标记基因集，用于预测样本中可能富集的细胞成分，是解析组织异质性的利器。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">7. TRR 开头的条目（例如 TRR01158）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 TRRUST 数据库。这是一个专门收录人类和小鼠转录因子调控关系的数据库，且所有数据均来自文献验证。它是探索上游分子机制的关键，能帮助研究者从基因表达变化推导出背后的关键转录因子。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">8. C 开头的条目（例如 C0221056）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 DisGeNET 数据库。这是一个目前整合最全的基因与人类疾病关联数据库。它通过文本挖掘和实验验证，建立了基因与疾病表型之间的联系，为基础研究成果提供了直接的临床转化线索。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">9. PGB 开头的条目（例如 PGB:00109）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 PaGenBase 数据库。这是一个组织特异性基因表达模式数据库。它定义了哪些基因在特定的组织或器官中特异性高表达，常被用于验证数据的组织来源准确性或探索基因的组织特异性功能。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">10. CORUM 数据库(例如 26S proteasome)</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 CORUM (The Comprehensive Resource of Mammalian protein complexes) 数据库。与前文提到的基于文献挖掘的 PPI 不同，CORUM 是专门收录哺乳动物蛋白质复合物的资源，且经过了人工注释。在富集分析中，如果基因列表富集到了 CORUM 条目，直接提示了这些基因可能共同参与了某个具体的蛋白复合物结构。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">11. P 开头的条目（例如 P00005）</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">对应 PANTHER Pathway 数据库。这是一个侧重于基因家族和进化关系的分类系统。它的通路图通常更简化和概括，覆盖了从信号转导到代谢的广泛生物学过程。在 ID 命名上，PANTHER 的通路通常以 </span><strong><span leaf="">P</span></strong><span leaf=""> 开头后跟数字编号。</span></p><h4 data-tool="mdnice编辑器"><span><span></span></span><span><span></span><span leaf="">Table 3.
