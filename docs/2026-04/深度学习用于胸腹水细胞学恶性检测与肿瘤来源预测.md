data-tool="mdnice编辑器"><span leaf="">胸水（pleural effusion）和腹水（ascites）细胞学检查是转移性肿瘤诊断中的重要手段。临床实践中，病理医师不仅需要判断体腔液中是否存在恶性肿瘤细胞，还希望在条件允许的情况下进一步推测肿瘤的原发来源。然而，传统显微镜阅片对阅片经验依赖较强，不同阅片者之间的一致性有限；对于细胞学阴性但临床仍高度怀疑恶性的病例，往往还需要结合免疫组化、流式细胞术、FISH 或组织活检等额外检查，增加了诊断成本和时间负担。对于晚期患者而言，体腔液细胞学材料有时已是最主要甚至唯一可获得的病理材料，因此，如何直接利用胸腹水细胞学切片实现更高效、更稳定的恶性检测与来源判断，具有明确的临床意义。</span></p><p data-tool="mdnice编辑器"><span leaf="">基于这一背景，本文提出了统一的深度学习框架 </span><strong><span leaf="">MAMILE-UNI</span></strong><span leaf="">，分别完成两项任务：一个是基于胸水和腹水的 WSI 进行恶性检测；另一个是在恶性基础上进一步开展肿瘤来源预测，并结合多中心 H&amp;E 病理切片来验证模型在癌源识别任务中的泛化能力。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">数据集信息</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">本文主要使用来自 </span><strong><span leaf="">Tri-Service General Hospital（TSGH）</span></strong><span leaf=""> 的胸水和腹水细胞学全切片图像评估模型的恶性检测能力，共纳入 </span><strong><span leaf="">1250 张 WSI</span></strong><span leaf="">，同时覆盖 </span><strong><span leaf="">cytology smear</span></strong><span leaf=""> 和 </span><strong><span leaf="">cell block</span></strong><span leaf=""> 两种制片形式，并且胸水和腹水两类样本都同时包含良性与恶性病例。</span></p><p data-tool="mdnice编辑器"><span leaf="">另一部分是来源于 </span><strong><span leaf="">TCGA</span></strong><span leaf=""> 的多中心病理数据集，用于验证模型在肿瘤来源预测任务中的跨中心泛化能力，它一共包含来自 69 个机构的 </span><strong><span leaf="">1196 张</span></strong><span leaf=""> H&amp;E WSI，主要覆盖支气管肺、乳腺、胃肠道和妇科来源肿瘤。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-tool="mdnice编辑器"><span><span leaf="">模型框架：MAMILE-UNI</span></span></h2><p data-tool="mdnice编辑器"><strong><span leaf="">1. Vision Foreground Extraction Module（VFEM）</span></strong><span leaf="">模型首先通过 VFEM 在 whole-slide image 中识别前景区域，即真正含有细胞或组织成分的部分，并尽量去除大面积空白背景。当前景区域确定后，作者再从这些区域中提取高倍率下的非重叠 patch，作为后续分析的输入。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">2. Multiple Instance Learning（MIL）</span></strong><span leaf="">将每张 WSI 视为一个 bag，bag 中包含多个 patch 实例。训练时只使用整张切片的标签，而不需要逐个 patch 的人工标注，因此 whole-slide classification 被转化为一个 MIL 问题。对于恶性检测任务，每张切片对应二分类标签；对于肿瘤来源预测任务，则对应多分类标签。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">3. Contrastive Transformer-based Feature Encoder（UNI）</span></strong><span leaf="">在完成前景提取和 MIL 任务设定后，作者使用 </span><strong><span leaf="">UNI</span></strong><span leaf=""> 作为特征编码器，将每个高维图像 patch 转换为低维特征向量。UNI 建立在 </span><strong><span leaf="">ViT-L/16</span></strong><span leaf=""> backbone 之上，并结合 </span><strong><span leaf="">DINOv2</span></strong><span leaf=""> 的 teacher-student 对比学习框架进行预训练。文中使用的编码器包含 </span><strong><span leaf="">24 个 transformer layers、16 个 attention heads</span></strong><span leaf="">，输出特征维度为 </span><strong><span leaf="">1024</span></strong><span leaf="">。经过这一过程，每张切片中的所有 patch 都会被表示为一组 1024 维特征向量，为后续聚合提供输入。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">4. Multilayered Attention-based Multiple Instance Learning（MAMIL）</span></strong><span leaf="">在获得 patch-level 特征后，MAMIL 进一步对这些特征进行加权聚合。先将每个 patch 的 1024 维特征进一步映射到 512 维表示，再通过注意力机制为每个 patch 赋予不同的重要性分数，最终对所有 patch 进行加权平均，得到 slide-level 特征向量。这样，模型能够在没有人工监督标出肿瘤区域的前提下，自动将更多权重放在诊断相关区域上。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">5. Early stop mechanism 与 ensemble voting</span></strong><span leaf="">在完成 MAMIL 聚合之后，进一步通过 early stopping 和 ensemble voting 提高预测稳定性。训练过程中以验证集交叉熵损失作为监控指标：当验证损失在连续 </span><strong><span leaf="">50 个 epoch</span></strong><span leaf=""> 内未继续下降时，触发 early stop；随后训练再继续 </span><strong><span leaf="">20 个 epoch</span></strong><span leaf="">，并从这一范围内选择验证损失最低的 epoch 作为最终模型。同时，最终结果并不依赖单个最优模型，而是基于不同训练-验证划分筛选出若干表现较好的模型，并通过 </span><strong><span leaf="">bagging</span></strong><span leaf=""> 的方式进行集成。最终预测结果由这些模型输出的阳性概率取平均得到，再根据预设阈值完成分类。</span></p><p data-tool="mdnice编辑器"><strong><span leaf="">6. Interpretability analysis</span></strong><span leaf="">在获得最终预测结果后，还进一步进行了可解释性分析。其基本思路是将 MAMIL 中注意力机制学习到的 patch 重要性重新映射回原始切片空间，从而生成 heatmap，观察模型主要关注哪些区域。这一步并不改变模型结构，而是用于解释模型在作出恶性检测或肿瘤来源预测时，主要依赖了哪些形态学区域，并进一步评估这些高注意力区域与病理医师关注区域之间的一致性。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img 的恶性检测任务中，MAMILE-UNI 整体优于其余五种对照方法。胸水数据上表现最好，尤其是 </span><strong><span leaf="">cell block</span></strong><span leaf="">，AUROC 达到 <span textstyle="">97%</span>，AUPRC 为 <span 与病理医师关注区域具有较好一致性，说明模型在一定程度上确实捕捉到了诊断相关区域，而不是仅依赖随机背景信息完成分类。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-tool="mdnice编辑器"><span><span leaf="">4. 数据效率与计算效率</span></span></h4><p data-tool="mdnice编辑器"><span leaf="">除分类性能外，MAMILE-UNI 还兼顾了数据效率和计算效率。与其他特征编码器相比，基于 </span><strong><span leaf="">UNI</span></strong><span leaf=""> 的版本在四个恶性检测任务中整体表现最好，同时在特征维度和存储开销上也较为平衡。推理速度方面，在低配 GPU 环境下，</span><strong><span leaf="">单张 WSI 的处理时间可控制在 2 分钟以内</span></strong><span leaf="">，而人工阅片通常至少需要 </span><strong><span leaf="">10 分钟</span></strong><span leaf="">。因此，这一框架不仅性能较高，也具备一定的临床部署潜力。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">总结</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">本文提出了一个统一的 whole-slide image 分析框架 </span><strong><span leaf="">MAMILE-UNI</span></strong><span leaf="">，并将其应用于胸水、腹水细胞学切片的恶性检测以及肿瘤来源预测任务。结果表明，该框架在胸水和腹水细胞学 WSI 的恶性检测中整体优于多种对照方法，其中 </span><strong><span leaf="">cell block</span></strong><span leaf=""> 的表现普遍优于 </span><strong><span leaf="">smear</span></strong><span leaf="">，胸水数据的表现也整体优于腹水数据；在此基础上，模型还进一步展现出一定的细胞学癌源预测能力，并在多中心组织病理数据上取得了更强的来源识别性能。这说明该方法不仅能够完成“是否恶性”的判断，也具备向“肿瘤来源推测”进一步拓展的潜力。</span></p><p data-tool="mdnice编辑器"><span leaf="">不过，这项工作仍然存在一些局限。首先，细胞学来源预测部分在不同类别之间的表现差异较大，对支气管肺和乳腺来源的识别相对较好，而对部分来源类别的区分能力仍然有限。其次，误差分析表明，模型在肿瘤细胞数量极少、形态不典型、存在明显伪影或反应性细胞干扰的病例中仍然容易出现误判。最后，尽管结果提示该框架具有一定临床应用前景，但目前的临床价值仍属于初步验证阶段，后续仍需要更多前瞻性研究和真实世界外部验证来进一步确认其稳健性与可推广性。</span></p></section><section><span
