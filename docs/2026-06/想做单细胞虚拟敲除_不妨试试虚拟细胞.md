想做单细胞虚拟敲除？不妨试试虚拟细胞 by 生信小博士
------
<div><h1 data-pm-slice="0 0 []"><span leaf="">开篇</span></h1><p><span leaf=""><span textstyle="">构建一个“虚拟细胞”——一个能够预测、模拟和编程细胞多模态、多尺度过程的计算系统——是计算生物学的长期愿景。</span>实现这一目标的一个重要里程碑是能够准确预测基因或化学扰动下的转录反应，这对于疾病机制研究和药物发现至关重要。</span></p><p><span leaf="">然而，当前的做法严重依赖人类专家：反复手工设计架构、调试代码、调超参数、跑实验，一个模型开发周期动辄数月。而且，<span textstyle="">由于扰动响应数据的高维性和生物学复杂性，人工设计的归纳偏置未必是最优的。</span></span></p><p><span leaf="">2026年4月，GenBio AI团队发布了一项突破性工作——</span><strong><span leaf=""><span textstyle="">VCHarness</span></span></strong><span leaf=""><span textstyle="">。这是一个自主AI系统，结合了多模态生物基础模型（AIDO系列）、AI编码智能体（Claude Sonnet 4.6）、蒙特卡洛树搜索（MCTS）和分布式执行，能够在</span></span><strong><span leaf=""><span textstyle="">几天内</span></span></strong><span leaf=""><span textstyle="">自动搜索出优于专家基线的扰动响应预测模型</span>，并发现非显而易见的架构设计模式。</span></p><p><span leaf=""><br></span></p><hr><h1><span leaf="">一、研究背景</span></h1><h2><span leaf="">虚拟细胞：从概念到构建</span></h2><p><span leaf="">“虚拟细胞”是近年来AI+生物学交叉领域的前沿概念：一个能够在统一框架内整合多模态、多尺度数据（基因组、转录组、蛋白质组、空间组学、病理影像等），并预测、模拟和编程细胞过程的计算系统。2024年，Bunne等人在</span><em><span leaf="">Cell</span></em><span leaf="">上发表了“How to build the virtual cell with AI”的路线图，提出了构建虚拟细胞的优先事项和机遇。</span></p><p><strong><span leaf="">核心：扰动响应预测</span></strong></p><p><span leaf="">在通往虚拟细胞的路上，一个明确且可验证的子任务是<span textstyle="">：</span></span><strong><span leaf=""><span textstyle="">给定一个遗传或化学扰动（如CRISPR敲除某个基因），预测基因表达的变化</span></span></strong><span leaf=""><span textstyle="">。这直接对应着药物靶点发现、机制研究和精准医学的核心需求。</span></span></p><h2><span leaf="">领域现状</span></h2><p><span leaf="">近年来，大量深度学习方法被用于扰动响应预测：</span></p><ul><li><p><strong><span leaf="">GEARS</span></strong><span leaf="">（Roohani et al., 2024）：利用基因共表达网络和图神经网络预测多基因扰动效应。</span></p></li><li><p><strong><span leaf="">单细胞基础模型</span></strong><span leaf="">（scGPT、scFoundation、Geneformer）：通过自监督学习预训练，再微调用于扰动预测。</span></p></li><li><p><strong><span leaf="">基于STRING PPI的GNN</span></strong><span leaf="">（Cole et al., 2026）：结合蛋白质互作网络显著提升性能。</span></p></li></ul><p><strong><span leaf="">然而，即使有了这些先进的构建模块，模型开发仍然严重依赖手动设计</span></strong><span leaf="">：</span></p><ul><li><p><span leaf="">什么样的编码器组合（DNA、RNA、蛋白、PPI）最有效？</span></p></li><li><p><span leaf="">微调策略是全部微调、部分微调还是LoRA？</span></p></li><li><p><span leaf="">如何融合多模态信息（拼接、注意力、门控）？</span></p></li><li><p><span leaf="">优化器、损失函数、调度器如何配置？</span></p></li></ul><p><strong><span leaf="">人工探索这些组合空间的效率极低</span></strong><span leaf="">，且容易陷入局部最优或依赖经验偏好。作者的核心论点正是：</span><strong><span leaf="">模型构建本身可以自动化</span></strong><span leaf="">——通过将“搜索模型架构”转化为“搜索可执行程序”，并利用现代AI智能体和树搜索算法实现。</span></p><hr><h1><span leaf="">二、科学假说</span></h1><blockquote><p><strong><span leaf="">通过将生物基础模型作为可复用组件、AI编码智能体作为程序生成器、蒙特卡洛树搜索作为探索策略，可以构建一个自主系统（VCHarness），在无需人工干预的情况下搜索可执行程序（完整训练流水线）空间，找到优于人工设计模型的扰动响应预测架构，并且这一过程在多个细胞系和扰动类型中可迁移。</span></strong></p></blockquote><p><span leaf="">换句话说，</span><strong><span leaf="">“设计模型”这件事本身可以被AI自动化，且效果更好、速度更快、发现更出乎意料</span></strong><span leaf="">。</span></p><hr><h1><span leaf="">三、VCHarness如何运作的呢？</span></h1><p><span
