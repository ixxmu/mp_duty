data-remoteid="" data-tool="mdnice编辑器"><span><span leaf="">18. 总结</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">在本期教程中，我们用Stephenson 2021 COVID-19单细胞BCR数据跑完了一套单细胞BCR + 转录组联合分析流程：</span></p><ul><li><section><span leaf="">原始数据包含5000个B细胞和24929个基因。</span></section></li><li><section><span leaf="">Chain QC后保留3278个single-pair BCR细胞。</span></section></li><li><section><span leaf="">exact CDR3定义得到3166个clonotypes，最大clone只有17个细胞。</span></section></li><li><section><span leaf="">SHM-aware hierarchical clustering得到3106个clones，最大clone增大到49个细胞。</span></section></li><li><section><span leaf="">Plasmablast和Plasma_cell_IgM/IgG/IgA中的扩增比例更高，提示抗原经历过的B细胞状态更容易出现克隆扩增。</span></section></li><li><section><span leaf="">Expanded cells在UMAP上集中于plasmablast、plasma cell和switched memory相关区域。</span></section></li><li><section><span leaf="">IGHV3-23、IGHV3-30、IGHV1-69、IGHV4-39等是主要IGHV基因。</span></section></li><li><section><span leaf="">CDR3长度主要集中在15到18 aa，没有明显异常长度偏移。</span></section></li><li><section><span leaf="">IgM和IgD的SHM较低，IgG和IgA的SHM更高，符合class switching和SHM共同发生的背景。</span></section></li><li><section><span leaf="">COVID-19严重程度分析显示，克隆扩增和IgG1比例在中等偏重状态更高，而不是在最严重状态一路升高。</span></section></li></ul><p
