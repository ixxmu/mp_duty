data-tool="mdnice编辑器"><span leaf="">单细胞数据整合是 scRNA-seq 分析里的常规步骤，只要数据来自不同样本、不同批次、不同平台，或者想把多个队列放在一起比较，通常都会先做 batch correction / integration。我们平时关注最多的是两个问题：一是不同 batch 是否混合得足够好，二是细胞类型的生物学差异有没有被保留下来。但这篇文章提出：</span><strong><span leaf="">如果不同数据集里的细胞类型组成原本就不一样，整合结果还可靠吗？</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">这里的数据不平衡不只是简单的细胞数多少不一样，而是包括三种情况：不同 batch 里出现的细胞类型不完全相同；同一种细胞类型在不同 batch 里的细胞数差异很大；不同细胞类型之间的比例明显不一致。这样的情况在真实数据比如发育数据、肿瘤样本、疾病队列、跨病人样本整合里其实非常常见。文章系统评估了这种 imbalance 对单细胞整合和下游分析的影响，并提出了 Iniquitate 这一扰动分析框架。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">Iniquitate：用扰动实验量化数据不平衡的影响</span></span></h2><p data-tool="mdnice编辑器"><strong><span leaf="">Iniquitate</span></strong><span leaf=""> 分析框架的整体流程是：先准备两个平衡的对照数据集，以及四个本身就存在复杂不平衡的真实数据集；然后使用五种常见的单细胞整合方法进行整合，包括 </span><strong><span leaf="">BBKNN、Harmony、Scanorama、scVI 和 Seurat</span></strong><span leaf="">；最后系统评估整合后的下游结果是否稳定，评估覆盖了聚类、差异表达、marker gene 注释、query-to-reference 映射、细胞类型分类和轨迹推断等常见分析任务。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-tool="mdnice编辑器"><span leaf="">图1b 提出了两个因素：</span><strong><span leaf="">aggregate cell type support</span></strong><span leaf=""> 和 </span><strong><span leaf="">minimum cell type center distance</span></strong><span leaf="">。aggregate cell type support 可以简单理解为某个细胞类型在所有 batch 里一共有多少细胞，细胞越少，整合结果越容易不稳定；minimum cell type center distance 可以理解为它和最相近细胞类型之间有多像，越相似，就越容易在降采样或缺失后被分到相近细胞类型里。图1c 则展示了作者提出的 balanced clustering metrics，用来修正传统指标在不平衡场景下容易被高比例细胞类型主导的问题。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">PBMC平衡数据中的扰动实验</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">首先从一个相对简单的 PBMC 数据集开始，这个数据集包含两个 batch，每个 batch 中都有相同的 6 类主要细胞类型，并且每种细胞类型都降采样到 400 个细胞，在实验起点上，两个 batch 的细胞类型组成、细胞数和比例都是平衡的。</span></p><p data-tool="mdnice编辑器"><span leaf="">接下来，随机选择一个 batch 和一个细胞类型，对该细胞类型进行两种扰动：一种是降采样到原来的 10%，另一种是完全 ablation，也就是直接移除，这样就可以人为构造不同程度的不平衡场景。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-src="https://mmbiz.qpic.cn/mmbiz_jpg/Go1sLc9TF8XMf3rGf9BUgsXUnXr87eD2yNc4ybdzKLy6EZdDKMYJ1mCadIbWg8lXE5Ph2jxcnmcCMPqVDMMHBhzHZmEU1bE5ibehxuQYl13Q/640?wx_fmt=other&amp;from=appmsg" data-tool="mdnice编辑器"><span leaf="">图2a-b 展示了扰动前的 PBMC 数据，细胞类型和 batch 都相对平衡。图2c 展示了扰动设计：随机选择一个 batch 和一个细胞类型，然后进行 10% downsampling 或 ablation。图2d 当某个细胞类型被降采样或删除后，该细胞类型在整合空间中的 KNN 分类 F1 score 会明显下降。这个现象在多种整合方法中都存在，说明它不是某一个方法的特殊问题，而是数据不平衡本身带来的系统性影响。</span></p><p data-tool="mdnice编辑器"><span leaf="">比较有意思的是，B cell 基本不受影响。作者认为这是因为 B cell 和其他 PBMC 细胞类型在转录组上相对更容易区分。为了验证这一点，作者进一步把原始细胞类型按相似性合并成更高层次的三类：B cell、monocyte 和 NK/T cell。合并之后再做扰动，性能下降就没有之前那么明显。这个结果说明，</span><strong><span leaf="">细胞类型之间的转录组相似性会放大数据不平衡带来的影响</span></strong><span leaf="">。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">数据不平衡会影响下游分析</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">文章接下来系统考察了数据不平衡对下游无监督聚类、marker gene、query-to-reference annotation 和轨迹推断的影响。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-src="https://mmbiz.qpic.cn/sz_mmbiz_jpg/Go1sLc9TF8Uic8hLqqSud2ZAEf5uaoFBlpeIX9aevFyjqniaCZ0CNDjc0WOyAraBAZs70LhcFEBQAtEPNgyuc8EDOia16xUAIpINynSMafuOsE/640?wx_fmt=other&amp;from=appmsg" data-tool="mdnice编辑器"><span><span leaf="">1. 无监督聚类</span></span></h3><p data-tool="mdnice编辑器"><span leaf="">理论上，在 PBMC 这个实验里，虽然某些 cell type 被降采样或删除了一个 batch 中的部分细胞，但整体数据中存在的细胞类型并没有改变，聚类数应该相对稳定，可结果并不是这样。图3a 显示，扰动之后 Leiden clustering 得到的 cluster 数目会出现明显变化，而且不同整合方法的变化模式还不一样。Scanorama 对不同细胞类型扰动都比较敏感；BBKNN 和 Seurat 在 CD8 T cell ablation 后变化更明显；Harmony 往往在扰动后增加 cluster number；scVI 则主要在 monocyte 相关扰动中出现变化。统计发现，细胞类型平衡扰动会导致整合后聚类数显著变化。</span></p><h3 data-tool="mdnice编辑器"><span><span leaf="">2. marker gene排序</span></span></h3><p data-tool="mdnice编辑器"><span leaf="">文章选取了已知 cell type marker，比较扰动前后 marker gene ranking 的变化。结果显示，在降采样或删除后，很多 marker gene 的排名会发生明显偏移，尤其是 CD8 T cell 和 CD14 monocyte 这类与其他细胞类型相对接近的群体。</span></p><h3 data-tool="mdnice编辑器"><span><span leaf="">3. query-to-reference映射</span></span></h3><p data-tool="mdnice编辑器"><span leaf="">文章使用 Seurat 4.0 的 reference mapping pipeline 进行测试。整体来看，大多数细胞类型的注释比较稳定，但 T cell 相关亚群表现出明显不稳定，尤其是 CD4 T cell 和 CD8 T cell，在扰动后容易被错误注释到其他 T cell 相关类别中。图3f-g 进一步展示了预测标签比例，可以看到 CD4/CD8 T cell 会被大量预测为 MAIT 或其他 T cell 细分类别。这说明 reference mapping 并不能天然消除 imbalance 的影响。如果 query 和 reference 之间细胞组成差异很大，尤其是相似细胞类型比例不同，映射和标签迁移也可能出现系统性偏差。</span></p><h3 data-tool="mdnice编辑器"><span><span leaf="">4. 轨迹推断</span></span></h3><p data-tool="mdnice编辑器"><span leaf="">PBMC 本身没有明显发育轨迹，所以文章使用了 mouse organogenesis 数据来测试。结果显示，即使 batch effect 本身不算特别强，细胞类型不平衡仍然会影响 pseudotime 的估计。尤其是 myocyte 被降采样或删除后，整合前后 pseudotime 的相关性下降最明显。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">肿瘤样本中的不平衡</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">作者进一步分析了一个 PDAC 数据集，包含 8 个 biopsy samples。这里把细胞分成了三个 compartment：</span><strong><span leaf="">epithelial normal、epithelial tumor 和 microenvironment</span></strong><span leaf="">。然后随机选择 4 个 batch，对某个 compartment 进行降采样或删除，再观察整合后 compartment classification 的变化。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-src="https://mmbiz.qpic.cn/mmbiz_jpg/Go1sLc9TF8Wcs4t2U2Qla2t7ZKRezRECwsn4GIG3ZnQAtFeGzLXxgWorcKh9nicjaeVlnD7NUavNaL6knLHicZQkGtAwBNu4tYhfyMYc4IW3Q/640?wx_fmt=other&amp;from=appmsg" data-tool="mdnice编辑器"><span leaf="">图4a 将正常导管/腺泡上皮细胞合并为 epithelial normal，恶性导管/腺泡细胞合并为 epithelial tumor，其余免疫和基质细胞合并为 microenvironment。图4b 显示不同 batch 中三个 compartment 的细胞数本来就很不均衡。图4c 则展示了扰动后不同 compartment 的 KNN 分类 F1 score。结果表明 microenvironment 被降采样或删除后，对整体分类影响相对较小；而 epithelial normal 和 epithelial tumor 之间更相似，因此当其中一个 compartment 被扰动时，对同类 compartment 的整合性能影响更明显。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">传统指标的优化</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">很多常用指标，比如 ARI、AMI、Homogeneity、Completeness、V-measure，本身并不会充分考虑细胞类型比例不平衡的问题。当一个数据集中某些细胞类型占比很高时，整体指标很容易被大类细胞主导；而少数细胞类型即使整合得很差，对总体分数的影响也可能不明显。</span></p><p data-tool="mdnice编辑器"><span leaf="">为了解决这个问题，作者提出了几种 </span><strong><span leaf="">balanced clustering metrics</span></strong><span leaf="">，这些指标的核心思想是：</span><strong><span leaf="">不要让高比例细胞类型主导评价结果，而是让每个 ground truth cell type 对最终分数有相对均等的贡献。</span></strong></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-src="https://mmbiz.qpic.cn/sz_mmbiz_jpg/Go1sLc9TF8WhaJm778NLVQ5orKddlsQE3S39h3y2Tn65SEFddicF4YZ439sa4kgbkl41nnDaf81jZol9vWxiazy5ySXKYZfricsbaxoRIqcwaU/640?wx_fmt=other&amp;from=appmsg" data-ratio="0.6194444444444445" data-type="other" data-w="1080" data-imgfileid="100057425" data-aistatus="1" src="https://mmbiz.qpic.cn/sz_mmbiz_jpg/Go1sLc9TF8WhaJm778NLVQ5orKddlsQE3S39h3y2Tn65SEFddicF4YZ439sa4kgbkl41nnDaf81jZol9vWxiazy5ySXKYZfricsbaxoRIqcwaU/640?wx_fmt=other&amp;from=appmsg"></span></figure><p data-tool="mdnice编辑器"><span leaf="">文章将 CD4 T cell 和 FCGR3A+ monocyte 在其中一个 batch 中降采样到原来的 10%，然后使用 BBKNN、Harmony、Scanorama 和 scVI 做整合，并比较传统指标和 balanced metrics 得到的排序。使用传统指标时，scVI 排名最高，BBKNN 排名最低；但使用 balanced metrics 后，Harmony 变成第一，BBKNN 也超过了 Scanorama。更重要的是，balanced metrics 能更清楚地区分 unintegrated embedding 和真正整合后的结果。作者认为，这说明 balanced metrics 能捕捉到传统指标忽略的、与细胞类型不平衡相关的细节。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">实际分析中应该怎么处理？</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">文章最后给出了一个 guideline，用来指导用户在不平衡数据整合中进行诊断和调整。</span></p><figure data-tool="mdnice编辑器"><span leaf=""><img data-src="https://mmbiz.qpic.cn/mmbiz_jpg/Go1sLc9TF8WQyC7fsQKsIkhkGPUiaql5RdO8nZ4tu4gvQgtzJicfDhWJDibSfribKiagOv07tAsQ61jFIlgvsrlicqwbInQOU29eOSggcT2dlq7WA/640?wx_fmt=other&amp;from=appmsg" data-ratio="0.5722222222222222" data-type="other" data-w="1080" data-imgfileid="100057424" data-aistatus="1" src="https://mmbiz.qpic.cn/mmbiz_jpg/Go1sLc9TF8WQyC7fsQKsIkhkGPUiaql5RdO8nZ4tu4gvQgtzJicfDhWJDibSfribKiagOv07tAsQ61jFIlgvsrlicqwbInQOU29eOSggcT2dlq7WA/640?wx_fmt=other&amp;from=appmsg"></span></figure><p data-tool="mdnice编辑器"><span leaf="">整个流程分成三个阶段：pre-integration、integration 和 post-integration。</span></p><p data-tool="mdnice编辑器"><span leaf="">在 </span><strong><span leaf="">pre-integration stage</span></strong><span leaf="">，首先应该在每个 batch 内部分别做无监督聚类，观察不同 batch 中 cluster 数量和比例是否稳定。如果有 reference atlas，也可以先对每个 batch 单独做 query-to-reference annotation，看不同 batch 中预测细胞类型和比例是否一致。如果这些结果已经提示明显 imbalance，就不能直接默认所有 batch 都应该被强行混合。</span></p><p data-tool="mdnice编辑器"><span leaf="">在 </span><strong><span leaf="">integration stage</span></strong><span leaf="">，需要根据数据特点选择合适的方法和参数。如果数据中存在明显的 unshared variation，比如肿瘤病人之间差异很大，或者发育数据中不同时间点细胞组成不同，就应该谨慎追求过强的 batch mixing。必要时可以调整方法参数，让模型更强调 biological heterogeneity preservation；对于具有时间顺序或发育结构的数据，也可以考虑 sequential integration。</span></p><p data-tool="mdnice编辑器"><span leaf="">在 </span><strong><span leaf="">post-integration stage</span></strong><span leaf="">，不能只看整合后的 UMAP 是否好看，还要比较整合前后的 cluster 是否合理，检查 batch mixing 是否过度，以及生物学异质性是否被保留。如果发现 batch correction 和 biological conservation 的 tradeoff 不合适，就需要回到 integration stage 重新调整。</span></p><h2 data-tool="mdnice编辑器"><span><span leaf="">总结</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">文章最重要的结论可以概括为三点。</span></p><p data-tool="mdnice编辑器"><span leaf="">第一，不平衡的影响不是随机噪声，而是有规律的。某个细胞类型越少，或者越接近其他细胞类型，它在整合过程中越容易受到影响。</span></p><p data-tool="mdnice编辑器"><span leaf="">第二，这种影响不是某一个整合方法特有的。作者测试了 BBKNN、Harmony、Scanorama、scVI 和 Seurat，结果显示 imbalance effect 在不同方法中都存在，只是表现形式不同。</span></p><p data-tool="mdnice编辑器"><span leaf="">第三，传统 benchmark 指标可能低估这种问题。因为很多指标容易被高比例细胞类型主导，所以需要 balanced metrics 或者更细粒度的 cell-type-specific 评估方式来发现少数细胞类型中的整合问题。</span></p><p data-tool="mdnice编辑器"><span leaf="">这篇文章对实际分析最直接的提醒是：做单细胞整合时，不要只看 UMAP 上 batch 是否混在一起，也不要只依赖一个整体分数判断方法好坏。尤其是在肿瘤、发育、疾病队列和多样本整合中，应该先检查每个 batch 的细胞组成差异，再结合具体生物学问题决定整合强度。否则，整合本身可能会把真正重要的生物学差异抹掉，或者把少数但关键的细胞类型错误合并到相近的大群体中。</span></p></section><section><span
