leaf="">当我们在做癌症相关的单细胞分析的时候，我们常常会想这么一件事，</span><strong><span leaf="">可不可以直接从单细胞转录组的数据中，推测出细胞的拷贝数变异情况？</span></strong></p><p data-tool="mdnice编辑器"><span leaf="">我们知道肿瘤细胞往往带有大片段染色体拷贝数改变，包括某条染色体臂整体扩增，或者一大片区域缺失。</span></p><p data-tool="mdnice编辑器"><span leaf="">单细胞RNA-seq虽然不是DNA测序，但是</span><strong><span data-tool="mdnice编辑器"><span leaf="">但这两个算法都有一个问题，他们是R语言写的，受限于R语言的因素，</span><strong><span leaf="">计算速度和内存占用在大规模图谱中都是一个较大的问题</span></strong><span leaf="">，虽然在几年前有InferCNVpy这个包，但实际上这个包只是inferCNV的简化实现，所以在omicverse此前的版本中，我一直没有整合这个有问题的包，因为我希望omicverse里的包都是最快最准确没有问题的。</span></p><p data-tool="mdnice编辑器"><span leaf="">在OmicVerse重构算法开源生态中，来自</span><strong><span leaf="">广州医科大学的冼杰燊</span></strong><span leaf="">将基于R语言的CopyKAT和InferCNV重写成基于Python的py-CopyKAT和py-InferCNV，我们将在本期教程中详细介绍如何在omicverse中使用这两个包。</span></p><ul><li><section><code><span leaf="">CopyKAT</span></code><span
