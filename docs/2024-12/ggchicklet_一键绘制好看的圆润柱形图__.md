data-tool="mdnice编辑器"><span></span><span>ggchicklet是什么？</span><span></span></h2><p data-tool="mdnice编辑器"><code><span>ggchicklet</span></code> 是一个基于 <code><span>ggplot2</span></code> 的 R 包，专门用于绘制圆润的柱形图（或称为“Chicklet Chart”）。它可以帮助用户创建更加视觉吸引且具有圆角效果的柱形图，并且提供更多的美观与定制选项。</p><ul data-tool="mdnice编辑器"><li><section><strong>「安装与加载」</strong></section></li></ul><p data-tool="mdnice编辑器">首先，确保已经安装并加载了 <code><span>ggchicklet</span></code> 包。如果尚未安装，可以通过以下命令进行安装：</p><pre data-tool="mdnice编辑器"><span></span><code><span># 安装 ggchicklet 包</span><br>install.packages(<span>"ggchicklet"</span>)<br></code></pre><p data-tool="mdnice编辑器">安装完成后，加载包：</p><pre data-tool="mdnice编辑器"><span></span><code><span># 加载 ggchicklet 包</span><br><span>library</span>(ggchicklet)<br></code></pre><ul data-tool="mdnice编辑器"><li><section><strong>「基本用法」</strong></section></li></ul><p data-tool="mdnice编辑器"><code><span>ggchicklet</span></code> 主要用于替代传统的 <code><span>geom_bar()</span></code> 或 <code><span>geom_col()</span></code>，通过添加圆角效果来改善柱形图的视觉效果。它的核心函数是 <code><span>geom_chicklet()</span></code>。</p><p data-tool="mdnice编辑器">以下是一个简单的例子，演示如何使用 <code><span>ggchicklet</span></code> 创建一个基本的圆润柱形图：</p><pre data-tool="mdnice编辑器"><span></span><code><span>library</span>(ggchicklet)<br><span>library</span>(hrbrthemes)<br><span>library</span>(ggplot2)<br><br>data(<span>"debates2019"</span>)<br><br>spkr_ordr &lt;- aggregate(elapsed ~ speaker, data = debates2019, sum)<br>spkr_ordr &lt;- spkr_ordr[order(spkr_ordr[[<span>"elapsed"</span>]]),]<br><br>debates2019$speaker &lt;- factor(debates2019$speaker, spkr_ordr$speaker)<br><br>ggplot(debates2019) +<br>  geom_chicklet(<br>    aes(speaker, elapsed, group = timestamp, fill = topic),<br>    colour = <span>"#252a32"</span><br>  ) +<br>  scale_y_continuous(expand = c(<span>0</span>, <span>0.01</span>), position = <span>"right"</span>) +<br>  ggthemes::scale_fill_tableau(<span>"Tableau 20"</span>) +<br>  coord_flip() +<br>  labs(x = <span>NULL</span>, y = <span>"Minutes Spoken"</span>, fill = <span>NULL</span>) +<br>  theme_ft_rc() +<br>  theme(panel.grid.major.y = element_blank()) +<br>  theme(legend.position = <span>"bottom"</span>)<br></code></pre><p data-tool="mdnice编辑器"><li><section><p><strong>「参数说明」</strong></p></section></li><li><section><p><code>fill</code>：设置柱形的填充颜色。</p></section></li><li><section><p><code>radius</code>：设置柱形的圆角半径。可以通过
