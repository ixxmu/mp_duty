data-tool="mdnice编辑器"><span><span leaf="">11. 总结</span></span></h2><p data-tool="mdnice编辑器"><span leaf="">在本期教程中，我们用Puram 2017头颈鳞癌单细胞数据，复现了Xiao 2019单细胞代谢图谱中的几个主要结论：</span></p><ul><li><section><span leaf="">使用 </span><code><span leaf="">ov.single.Metabolism(method='scmetabolism')</span></code><span leaf=""> 计算了5,578个细胞的85条KEGG代谢通路活性。</span></section></li><li><section><span leaf="">恶性细胞中有53条KEGG代谢通路显著上调，是所有细胞类型里最多的。</span></section></li><li><section><span leaf="">氧化磷酸化在肿瘤细胞内部并不均匀，提示恶性细胞内部存在代谢异质性。</span></section></li><li><section><span
