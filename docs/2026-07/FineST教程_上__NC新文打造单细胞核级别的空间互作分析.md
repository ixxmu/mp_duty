FineST教程（上）：NC新文打造单细胞核级别的空间互作分析 by 生信宝库
------
<div><section data-pm-slice="0 0 []"><section><span><span leaf="">生信文献解读</span></span></section><h2><span leaf="">写在前面</span></h2><p><span leaf="">空间转录组正在从“看见组织位置”走向“解析细胞互作”。但对 10x Visium 这类 spot-level 数据来说，一个 spot 往往混合多个细胞，RNA 捕获又比较稀疏，直接做 ligand-receptor 分析时，很容易把真实的细胞级互作信号稀释掉。</span></p><p><span leaf="">2026 年 Nature Communications 发表的 FineST 就是为了解决这个问题。文章题为 </span><strong><span leaf="">FineST: contrastive learning integrates histology and spatial transcriptomics for nuclei-resolved ligand-receptor analysis</span></strong><span leaf="">，由 Lingyu Li、Tianjie Wang、Zhuo Liang、Huajian Yu、Stephanie Ma、Lequan Yu 和 Yuanhua Huang 等作者完成。</span></p><p><span leaf="">FineST 的核心思路很直接：不要只看表达矩阵，也不要浪费已经随空间转录组一起获得的 H&amp;E 图像。它用组织病理图像 foundation model 提取图像特征，再通过对比学习把 histology 和 spatial transcriptomics 融合起来，最终把 spot-level 表达推到 sub-spot 或 single-nucleus 级别，用于更细粒度的 ligand-receptor 和 cell-cell communication 分析。</span></p><section nodeleaf=""><img data-aistatus="1" data-src="https://mmbiz.qpic.cn/mmbiz_png/I7BPRRicNMlTsObQksgnojXy6pZwMp69UAtdqCPu3yiaBibnL5zOf3IFlTYFPNPsWvR83o9RfLeb8huic953CiaiabNcAF2sP5iaFqDBP0jB6Q1oSk/640?wx_fmt=png&amp;from=appmsg" leaf=""><br></span></p><h2><span leaf="">FineST 解决的核心痛点</span></h2><p><span leaf="">传统空间转录组分析有一个矛盾：我们想研究的是细胞与细胞之间的通信，但 Visium spot 的物理尺度往往高于单细胞尺度。一个 spot 里可能有肿瘤细胞、成纤维细胞、免疫细胞和血管相关细胞混在一起。</span></p><p><span leaf="">这会带来三个问题。</span></p><p><span leaf="">第一，配体和受体的空间共表达不够细。spot-level 共表达不一定代表真正相邻细胞之间发生互作，也可能只是混合信号。</span></p><p><span leaf="">第二，低表达基因更容易丢失。很多 ligand-receptor gene 本身表达不高，在稀疏矩阵里容易被 dropout 掩盖。</span></p><p><span leaf="">第三，肿瘤微环境中关键结构往往很局部。比如 invasive front、TLS、免疫治疗抵抗屏障，真正有意义的区域可能只占组织切片的一小部分。</span></p><p><span leaf="">FineST 的定位，就是把空间转录组从 spot-level 推向更精细的空间单位，让后续的基因表达插补、细胞类型预测和配体受体分析更贴近组织真实结构。</span></p><h2><span leaf="">方法框架：Training、Imputation、Discovery</span></h2><p><span leaf="">根据 GitHub README，FineST 在 H&amp;E 图像特征提取之后，主要分成三个模块。</span></p><p><strong><span leaf="">Step 0：H&amp;E image feature extraction</span></strong></p><p><span leaf="">FineST 先从组织图像中提取 patch-level 图像特征。官方教程提供两条路线：HIPT 和 Virchow2。HIPT 不需要 Hugging Face token，适合快速入门；Virchow2 是更强的病理 foundation model，但通常需要申请模型权限。</span></p><p><strong><span leaf="">Step 1：Training FineST on within spots</span></strong></p><p><span leaf="">在已有 spot 内，FineST 学习图像特征和空间 RNA 表达之间的映射。可以理解为：先在已知位置上学会“什么样的组织图像局部结构，对应什么样的表达模式”。</span></p><p><strong><span leaf="">Step 2：Super-resolution RNA imputation</span></strong></p><p><span leaf="">训练完成后，FineST 可以把表达插补到更高分辨率。对 Visium 数据，可以插补到 sub-spot；结合 nuclei segmentation 时，可以进一步做到 single-cell 或 nuclei-resolved 层面。对 Visium HD 这类连续 bin-squares 数据，则可以直接围绕更小 bin 做图像特征和表达建模。</span></p><p><strong><span leaf="">Step 3：Fine-grained ligand-receptor discovery</span></strong></p><p><span leaf="">最后，FineST 用更细粒度的表达矩阵和空间位置去发现 ligand-receptor pair 以及局部 cell-cell communication pattern。这个步骤是整篇文章和软件名字里 “Fine-grained Spatial Transcriptomics” 的落脚点。</span></p><section nodeleaf=""><img data-aistatus="1" data-imgfileid="100015254" data-ratio="0.44820065430752454" data-s="300,640" leaf=""><br></span></p><h2><span leaf="">论文验证：从 CRC、Xenium 到多癌种 Visium</span></h2><p><span leaf="">论文摘要中提到，作者用 colorectal cancer Visium HD 和 breast cancer Xenium 数据验证 FineST。在这些数据上，FineST 在高分辨率 RNA expression imputation、cell type prediction 和 CCC pattern discovery 中优于已有方法。</span></p><p><span leaf="">这类验证很关键。因为 FineST 不是单纯做漂亮 UMAP，而是要回答三个更实际的问题。</span></p><p><span leaf="">第一，插补后的表达是否更接近高分辨率或单细胞参考？</span></p><p><span leaf="">第二，插补表达能否帮助更准确地定位细胞类型？</span></p><p><span leaf="">第三，更细粒度的表达是否真的能发现更有生物学意义的 ligand-receptor pattern？</span></p><p><span leaf="">FineST 的目标不是替代所有空间分析工具，而是补上 spot-level 数据到 fine-grained communication analysis 之间的分辨率缺口。</span></p><section nodeleaf=""><img data-aistatus="1" data-imgfileid="100015256" data-ratio="0.7464940668824164" leaf=""><br></span></p><h2><span leaf="">肿瘤微环境中的应用场景</span></h2><p><span leaf="">FineST 特别适合肿瘤微环境研究，原因是肿瘤组织中很多关键过程都发生在很局部的空间边界上。</span></p><p><strong><span leaf="">场景一：肿瘤侵袭前沿</span></strong></p><p><span leaf="">乳腺癌、结直肠癌和肝癌等实体瘤中，invasive front 往往是肿瘤细胞、基质细胞、髓系细胞和 T 细胞相互作用最活跃的区域。FineST 可以帮助把 spot-level 的混合信号拆得更细，观察某些 ligand-receptor pair 是否集中在侵袭边界。</span></p><p><strong><span leaf="">场景二：三级淋巴结构 TLS</span></strong></p><p><span leaf="">TLS 研究强调 B cell、T cell、DC、FDC-like stromal cells 等细胞之间的空间组织关系。FineST 在 nasopharyngeal carcinoma 等场景中展示了对 TLS 相关互作的解析潜力。对做肿瘤免疫的人来说，这很适合和 TLS score、B cell follicle marker、CXCL13/CXCR5 等信号结合分析。</span></p><p><strong><span leaf="">场景三：免疫治疗抵抗屏障</span></strong></p><p><span leaf="">论文摘要提到，FineST 用于 hepatocellular carcinoma 的 PD-1 therapy resistance barrier 分析。这里的重点不是简单比较 responder 和 non-responder，而是定位哪些空间区域可能形成免疫排斥、髓系抑制或 stromal barrier，从而影响 T cell 进入和功能维持。</span></p><section nodeleaf=""><img leaf=""><br></span></p><h2><span leaf="">和常规空间分析流程怎么衔接</span></h2><p><span leaf="">如果已经有一套 Seurat/Scanpy/Squidpy/CellChat/CellPhoneDB 的空间分析流程，FineST 更像是插在前处理和通信分析之间的一个“分辨率增强模块”。</span></p><p><span leaf="">一个比较自然的分析路线是：</span></p><p><span leaf="">1. 用 Space Ranger 或平台官方流程生成表达矩阵、空间坐标和 H&amp;E 图像。 2. 对空间数据做基础 QC、标准化、空间可视化和粗粒度聚类。 3. 用 FineST 提取 H&amp;E 图像特征，训练图像-表达映射模型。 4. 生成 sub-spot 或 nuclei-resolved 的高分辨率表达矩阵。 5. 基于插补表达进行 cell type prediction、marker 可视化和空间区域注释。 6. 对感兴趣区域做 ligand-receptor 和 cell-cell communication 分析。 7. 回到原始图像和组织结构中解释结果，避免只看统计显著性。</span></p><p><span leaf="">这个流程最适合回答“某个组织结构里发生了什么互作”这类问题，而不是单纯给全切片做一个平均通信网络。</span></p><h2><span leaf="">局限性和注意事项</span></h2><p><span leaf="">第一，FineST 依赖组织图像质量。H&amp;E 图像模糊、组织折叠、染色差异过大、切片区域不完整，都会影响图像特征质量。</span></p><p><span leaf="">第二，插补不是实测。FineST 预测的是更高分辨率表达，不能把插补结果当成真正的单细胞测序结果。关键发现仍然需要用原始 spot 表达、空间位置、免疫荧光、RNAscope 或独立数据验证。</span></p><p><span leaf="">第三，模型路线对计算资源有要求。HIPT 路线更适合快速上手，Virchow2 路线更接近论文主线，但可能需要 Hugging Face 模型权限和更强 GPU。</span></p><p><span leaf="">第四，ligand-receptor 分析要谨慎解释。空间共表达和邻近性可以提出互作假设，但不能直接证明功能性信号传递。最好结合 receptor downstream pathway、细胞状态、扰动实验或治疗响应数据来解释。</span></p><h2><span leaf="">小结</span></h2><p><span leaf="">FineST 的价值在于把空间转录组、组织病理图像和细胞通信分析串成一条完整路线。它不是只做图像，也不是只做表达插补，而是把 H&amp;E 中的组织结构信息用到空间表达重建中，再服务于 nuclei-resolved ligand-receptor analysis。</span></p><p><span leaf="">对做肿瘤空间组学的人来说，FineST 尤其适合三类问题：侵袭边界的局部互作、TLS 或免疫生态位的精细解析、免疫治疗抵抗相关的空间屏障发现。下一篇我们会进入代码实操，从安装环境、下载教程数据到跑通 HIPT demo 开始。</span></p><h2><span leaf="">参考资料</span></h2><p><span leaf="">1.
