lang="EN-US"><span leaf="" mpa-font-style="mrofsfokmtt" data-mpa-action-id="mrofsfor1az1" data-pm-slice="0 0 []"><span textstyle="">一、研究背景与现有痛点</span></span></span></h2><p><span lang="EN-US"><span leaf="">药物研发正面临"反摩尔定律"困境——投入成本持续攀升，但获批新药数量却不增反降。在这个背景下，利用人工智能预测化学药物对细胞的影响，成为早期药物筛选的核心方向之一。</span></span></p><p><span lang="EN-US"><span leaf="">问题的关键在于：同一种药物在不同细胞类型中会引发完全不同的反应。比如，免疫调节剂 IFNβ 对 B 细胞和 T 细胞的转录影响可能存在显著差异。然而，由于高通量单细胞药物实验成本高昂、技术复杂，现有的实验数据往往只覆盖了极少数的药物和细胞类型组合——绝大多数实验仅测量数十种药物在少数细胞类型中的反应。</span></span></p><p><span lang="EN-US"><span leaf="">已有的方法主要分为两类：一类是通过生成模型（如 scGen、chemCPA）将数据映射到低维隐空间后做向量线性外推，但这类方法需要大量数据来学习分布参数，且在训练数据之外的细胞类型上泛化能力有限；另一类是利用基因-基因互作网络作为先验知识来预测，但这些方法忽略了不同细胞类型之间的特异性差异——它们为所有细胞类型使用同一张基因交互图，抹平了关键的细胞类型特征。</span></span></p><p><span lang="EN-US"><span leaf="">用一个通俗的比喻：传统方法像是一位医生，用同一套诊断标准去看所有病人；而 PrePR-CT 的做法则是根据不同病人的体质特点（细胞类型），为每位病人建立专属的"病历档案"（细胞类型特异性共表达网络），从而做出更精准的判断。</span></span></p><h2><span lang="EN-US"><span leaf="" mpa-font-style="mrofsp8316d3" data-mpa-action-id="mrofsp8a18ra" data-pm-slice="0 0 []"><span textstyle="">二、核心创新点</span></span></span></h2><p><span lang="EN-US"><span leaf="">PrePR-CT 的核心创新可以归纳为"一个先验、两重机制"：</span></span></p><p><span lang="EN-US"><span><span leaf="">·</span></span></span><span lang="EN-US"><span leaf="">1. </span><b><span leaf="">细胞类型特异性共表达网络作为归纳先验</span></b><span leaf="">：PrePR-CT 不依赖大量训练数据，而是利用"细胞类型"本身携带的生物信息作为模型的先验知识。具体而言，它为每种细胞类型构建一个基因共表达图谱，基因是图中的节点，表达模式高度相关的基因之间连上边。这个图谱刻画了该细胞类型的"分子指纹"，模型借此理解不同基因在这种细胞类型中如何协同工作。</span></span></p><p><span lang="EN-US"><span><span leaf="">·</span></span></span><span lang="EN-US"><span leaf="">2. </span><b><span leaf="">图注意网络学习细胞类型表征</span></b><span leaf="">：PrePR-CT 使用图注意网络（GAT）来从共表达图谱中提取特征，赋予每个基因节点一个"注意力分数"，反映该基因在当前细胞类型中的重要性。这种机制不仅提升了预测精度，还实现了可解释性——研究者可以通过注意力分数追溯模型做出某个预测时最看重哪些基因。</span></span></p><p><span lang="EN-US"><span><span leaf="">·</span></span></span><span lang="EN-US"><span leaf="">3. </span><b><span leaf="">小数据条件下的鲁棒性能</span></b><span leaf="">：通过引入生物学先验，PrePR-CT 在数据极度有限的场景下（仅使用20%的训练细胞）仍能维持有效预测，明显优于依赖大量数据拟合的生成模型。</span></span></p><h2><span lang="EN-US"><span leaf="" mpa-font-style="mroftb5we1n" data-mpa-action-id="mroftb63gft" data-pm-slice="0 0 []"><span textstyle="">三、具体方法</span></span></span></h2><h3><span lang="EN-US"><span leaf="" mpa-font-style="mrofvblm1885" data-mpa-action-id="mrofvblt19h" data-pm-slice="0 0 []"><span textstyle="">3.1 模型基础理论：什么是图注意网络？</span></span></span></h3><p><span lang="EN-US"><span leaf="">PrePR-CT 的核心技术基础是图注意网络（Graph Attention Network，GAT）。与传统的卷积神经网络（CNN）擅长处理网格化数据（如图片）不同，GAT 是为处理图结构数据而设计的。</span></span></p><p><span lang="EN-US"><span leaf="">如果把每个细胞类型看作一个社交网络，基因就是网络中的人，边表示两人之间关系密切（共表达）。GAT 的工作方式好比：在判断一个人的重要性时，不仅要看他自己的特征，还要看他与哪些朋友密切联系，以及这些朋友对他的影响力有多大。通过注意力机制，GAT 为每个基因的每个邻居自动计算一个"影响力权重"，将有影响力的邻居信息聚合到该基因的特征表示中。</span></span></p><p><span lang="EN-US"><span leaf="">这种机制特别适合 PrePR-CT 的场景：不同细胞类型中，基因之间的"社交关系"完全不同，而 GAT 的多头注意力机制可以同时从多个角度观察这些关系，捕捉到丰富的生物学信号。</span></span></p><h3><span lang="EN-US"><span leaf="" mpa-font-style="mrofv7fc9uz" data-mpa-action-id="mrofv7fjhvm" data-pm-slice="0 0 []"><span textstyle="">3.2 模型架构设计</span></span></span></h3><p><span lang="EN-US"><span leaf="">PrePR-CT 的整体工作流程可以用"三步走"来概括：</span></span></p><p><b><span lang="EN-US"><span leaf="">第一步：构建细胞类型图谱。</span></span></b><span lang="EN-US"><span leaf=""> 对每个细胞类型，研究者利用未受干扰（对照）状态下的单细胞数据，通过 SEACells 算法将相似细胞聚合成"元细胞（metacell）"，以消除单细胞数据的稀疏噪音。随后在元细胞层面计算5000个高变基因之间的两两相关性，仅保留相关性排名前1%的边，形成每个细胞类型的专属共表达网络。</span></span></p><p><b><span lang="EN-US"><span leaf="">第二步：图注意网络提取特征。</span></span></b><span lang="EN-US"><span leaf=""> 每个基因节点的初始特征由对照状态下的表达均值和标准差组成。通过多层 GAT 处理后，每个基因获得一个包含其邻居信息的高维特征向量。然后对图中所有节点做最大池化，得到一个代表整个细胞类型的特征向量。</span></span></p><p><b><span lang="EN-US"><span leaf="">第三步：联合预测转录反应。</span></span></b><span lang="EN-US"><span leaf=""> 将细胞类型特征向量、药物分子嵌入向量（由 SMILES 化合物结构通过 RDKit 生成的化学描述符）以及对照细胞的基因表达值拼接后，输入多层感知机（MLP）回归器，最终输出每个基因在受药物干扰后的表达预测值。</span></span></p><p><span lang="EN-US"><span leaf="">值得注意的是，PrePR-CT 使用了 Wasserstein 距离（推土机距离）而非传统的均方误差（MSE）作为损失函数。Wasserstein 距离衡量的是两个分布之间实现转换所需的最小"搬运成本"，相比 MSE 能更好地捕捉预测表达分布与真实分布之间的整体差异，对低表达基因的预测尤其有益。</span></span></p><h3><span lang="EN-US"><span leaf="" mpa-font-style="mrofvnr12x2" data-mpa-action-id="mrofvnr911l4" data-pm-slice="0 0 []"><span textstyle="">3.3 训练与评估设计</span></span></span></h3><p><span lang="EN-US"><span leaf="">研究团队在5个单细胞 RNA 测序数据集和1个大规模药物筛选数据集上进行了全面评估，涵盖人外周血单个核细胞（PBMC）、多种人类癌细胞系、小鼠肝脏等不同生物背景。评估采用"留一细胞类型交叉验证"——训练时完全排除测试细胞类型的共表达图谱，测试模型的跨细胞类型泛化能力。</span></span></p><p><span lang="EN-US"><span leaf="">主要的对比方法包括 scGen、CPA（单药物场景）和 chemCPA（多药物场景），评估指标为决定系数 R²，同时关注全部基因和前100个差异表达基因（DEGs）的预测表现。</span></span></p><h2><span lang="EN-US"><span leaf="" mpa-font-style="mrofvs5c14ql" data-mpa-action-id="mrofvs5jc7" data-pm-slice="0 0 HAG 分析的稳定性受训练参数（如注意力头数）的影响，需要在不同条件下评估其一致性。</span></span></p><p><span lang="EN-US"><span><span leaf="">·</span></span></span><span lang="EN-US"><span leaf="">4. 在严格分布外场景（测试细胞类型的对照细胞完全不参与训练）中，简单的均值基线有时能超过深度模型，这呼应了近期多项基准研究所揭示的"深度学习模型未必优于线性基线"的问题。</span></span></p><h2><span lang="EN-US"><span leaf="" mpa-font-style="mrog4ut964i" data-mpa-action-id="mrog4utdzhx" data-pm-slice="0 0 []"><span textstyle="">七、未来方向</span></span></span></h2><p><span lang="EN-US"><span leaf="">作者提出了几个有前景的扩展方向：将方法扩展到预测药物组合效应、识别治疗靶点、探索不同药物扰动之间的交互作用。更长远地看，这种"嵌入细胞特异性系统生物学模型"的思路，可能为从头构建定量化、可预测的细胞生物学模型铺平道路。</span></span></p><h2><span lang="EN-US"><span leaf="" mpa-font-style="mrog50wkbmm" data-mpa-action-id="mrog50wo1mbp" data-pm-slice="0 0 []"><span textstyle="">结语</span></span></span></h2><p><span lang="EN-US"><span leaf="">PrePR-CT 的核心启示在于：好的归纳先验胜过海量数据。通过为每种细胞类型构建专属的基因共表达"社交网络"，并利用图注意网络从中挖掘关键信息，这一方法在数据有限的现实场景中为药物反应预测提供了新范式。它不仅预测得准，还能解释为什么准——这在将 AI 真正用于药物发现的过程中，或许是最关键的一步。</span></span></p><p><b><span lang="EN-US"><span leaf=""><span textstyle="">参考文献</span></span></span></b></p><p><span lang="EN-US"><span leaf="">Alsulami R, Lehmann R, Khan SA, Lagani V, Maillo A, Gomez-Cabrero D, Kiani NA, Tegner J.
